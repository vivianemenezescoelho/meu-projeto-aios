## Configurações Recomendadas

### Para Midjourney:
- Aspect ratio: `--ar 16:9` (ideal para reels/vídeos)
- Versão: `--v 6` ou `--v 6.1`
- Estilo: `--style raw` para menos estilização

### Para DALL-E 3:
- Tamanho: 1792x1024 (widescreen)
- Qualidade: HD

### Para Leonardo AI:
- Modelo: Leonardo Diffusion XL
- Guidance Scale: 7-8
- Aspect ratio: 16:9

---

## Dica de Consistência Visual

Para manter unidade visual no post:
1. Use a mesma paleta de cores (azul tech + tons quentes)
2. Mantenha mesmo estilo de iluminação (ambiente + pontos de destaque)
3. Considere adicionar overlay sutil com gradiente nas imagens
4. Use a mesma tipografia nos textos sobre imagem
